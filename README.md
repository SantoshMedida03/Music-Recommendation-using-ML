# Music-Recommendation-using-ML
A hybrid music recommendation system that suggests songs based on features, mood, and popularity filtering. Built using Python, Flask, Pandas, NumPy, and Scikit-learn, the system provides an interactive web interface where users can explore recommendations and give feedback for adaptive learning.

A **hybrid music recommendation system** that suggests songs based on **features, mood, and popularity filtering**. Built using **Python, Flask, Pandas, NumPy, and Scikit-learn**, the project provides a web interface where users can explore personalized recommendations and give feedback to improve the model.  

---

## 🚀 Features  
- 🎵 **Hybrid Model** – Feature-based, mood-based, and popularity-based filtering.  
- 🌐 **Interactive Web App** – User-friendly interface powered by Flask.  
- 🔄 **Feedback Mechanism** – Learns and adapts to user preferences.  
- 📊 **Machine Learning** – Clustering & similarity measures for personalization.  

---

## 🛠️ Tech Stack  
- **Backend:** Python, Flask  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Frontend:** HTML, CSS (Flask templates)
